## How to Contribute

### Getting Started

1. **Fork the repository** - Click the "Fork" button at the top right
2. **Clone your fork** - `git clone https://github.com/YOUR_USERNAME/My-ML-Projects.git`
3. **Create a branch** - `git checkout -b your-feature-name`
4. **Make your changes** - Add code, fix bugs, improve documentation
5. **Commit your changes** - `git commit -m "Add: your meaningful commit message"`
6. **Push to your fork** - `git push origin your-feature-name`
7. **Open a Pull Request** - Go to the original repo and click "New Pull Request"
